🔧 How it works:

  1. When traveler visits /traveler/{token}, the component loads trip data
  2. Trip data includes blog_language (e.g., "de" for German)
  3. useEffect automatically calls i18n.changeLanguage(trip.blog_language)
  4. All UI text switches to the selected language
  5. Falls back to English if language not available
